# 🌙 Luna-OS 🐱‍🚀
**Kernel Mode Gaming Operating System**

## What's Working So Far? ✅
- **Memory allocation with paging.** 🧠  
- **32-bit 2D and 3D graphics** (via MicroGL and PortableGL) 🎨  
- **Standard C library functions.** 📚
